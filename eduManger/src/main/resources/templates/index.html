프로젝트 명

LMS 기반의 오프라인 학원 관리 시스템입니다.
프로젝트 목표

강사, 강의, 학생 정보의 효과적인 관리를 통해 학원관리자의 학원 운영의 효율성을 끌어올릴 수 있도록 프로그램을 제공합니다.
기대효과

학원관리자는 강의, 강의실, 강사 운영, 수납 및 납부 현황을 강사는 강의 현황을 학생은 수강과목 현황을 이해하기 쉬운 도식화된 데이터로 확인할 수 있습니다. 또한 학생은 성적데이터를 입력해 내신을 보다 편리하게 관리하며 출결데이터 확인으로 수업 내,외의 효과적인 시간배분을 통해 학습에 주도적으로 임할 수 있습니다. 또한 학부모는 학생의 출결데이터를 이용해 귀가시간 등을 전달받아 심야 귀가의 안정성 을 확보 할 수 있습니다.
개발환경

OS: Windows7
Client Language: JavaScript, HTML5, CSS, jsp 2.2, jQuery
Server Language: Java, JavaScript
WAS: Apache Tomcat
DB: MariaDB
