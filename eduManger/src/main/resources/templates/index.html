프로젝트 소개


LMS 기반의 오프라인 학원 관리 시스템입니다.


팀 프로젝트

교육과정 : 스마트웹 개발자 양성과정 (한국스마트 정보 교육원, 2019.05 ~ 2019.11)
기간 : 2019.08.12 ~ 2019.10.23 (총 72일)
팀원 : 김광준, 백X산, 이X윤, 이X율
역할 : 팀장
담당업무 : 학원관리 - 수납 및 납부안내, 수강료 및 할인 정보, 직원급여관리, 학원정보관리, 문자발송 / 계약 - 계약목록, 계약단가, 계약지점

개발환경
OS: Windows7
Client Language: JavaScript, HTML5, CSS, jsp 2.2, jQuery
Server Language: Java, JavaScript
WAS: Apache Tomcat
DB: MariaDB



오른쪽 상단의 로그인 버튼을 누르고 권한별 버튼을 누르면 해당권한으로 자동 로그인 됩니다
계약부분은 최고관리자로 로그인하시면 수정, 삭제버튼 활성화 됩니다.
